Virtualization Labs
	1. Install a Type-2 Hypervisor
		○ Tools: VMware Workstation or Oracle VirtualBox
		○ Lab: Create multiple VMs (Windows + Linux), configure networking between them.
		○ Skills: VM creation, snapshots, resource allocation.
	2. Hyper-V Lab
		○ Tools: Windows 10/11 Pro or Server
		○ Lab: Enable Hyper-V, create VMs, set up virtual switches.
		○ Skills: Windows virtualization, networking.
	3. Nested Virtualization
		○ Lab: Run a hypervisor inside a VM (e.g., ESXi inside VMware Workstation).
		○ Skills: Advanced virtualization concepts.
	4. VMware ESXi Lab
		○ Tools: ESXi on bare metal or nested
		○ Lab: Deploy ESXi, create VMs, configure vSwitches.
Skills: Enterprise virtualization basics. 

Container Labs
	1. Docker Basics
		○ Lab: Install Docker, run hello-world, deploy NGINX container.
		○ Skills: Images, containers, ports.
	2. Build Your Own Image
		○ Lab: Create a Dockerfile for a simple web app.
		○ Skills: Image building, layering.
	3. Docker Compose
		○ Lab: Deploy a multi-container app (e.g., WordPress + MySQL).
		○ Skills: Compose files, service linking.
	4. Kubernetes Intro
		○ Tools: Minikube or Kind
		○ Lab: Deploy pods, services, and scale an app.
		○ Skills: Orchestration, YAML configs.
	5. Networking & Volumes
		○ Lab: Create custom Docker networks, attach volumes for persistent storage.
		○ Skills: Container networking, data persistence.
